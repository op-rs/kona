# P2P RPC Methods

The `p2p` api provides methods for interacting with Kona's P2P stack.

## `opp2p_self`

Returns information about the local node in the form of `PeerInfo`.

| Client | Method invocation          |
| ------ | -------------------------- |
| RPC    | `{"method": "opp2p_self"}` |
